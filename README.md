# 🖥️ Project - Godot Computer Screen

An interactive terminal system developed in **Godot 4**, simulating a vintage CRT monitor for a psychological horror and investigation RPG experience.

## About
This system serves as the primary interface for players to navigate confidential files, monitor high-interest subjects, and analyze evidence related to the **Umbral** and the **Keys**.

### Features:
* **Multi-Page Navigation:** Seamlessly switch between Logs, Monitoring, and Attachments.
* **Dynamic Typewriter Effect:** Text renders in real-time with synchronized binary sound effects.
* **Evidence Gallery:** Supports image and document viewing with built-in pagination.
* **Retro Aesthetic:** High-contrast monochromatic interface featuring scanlines and CRT shaders.

## Tech Stack
* **Engine:** Godot 4.x
* **Modeling:** Blender
* **Language:** GDScript
* **Styling:** Advanced BBCode (RichTextLabel) and custom GLSL Shaders.
